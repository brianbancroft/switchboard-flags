## Source of truth

GitHub is the primary external collaboration link for this project.

## Suggested flow

1. Create a branch for the change.
2. Update code and docs together when behavior changes.
3. Open a pull request for review.
4. Use the docs site to capture architecture and workflow decisions that should persist.

## Docs updates

When a new feature changes developer setup, API contracts, or architecture boundaries, add or refresh the relevant page in this docs app as part of the same change.
