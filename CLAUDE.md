## Package manager

This repo uses **pnpm 9**. Always use `pnpm` — never `npm` or `yarn`.

## Commands

Run from the repo root unless targeting a specific workspace.

| Task | Command |
|------|---------|
| Dev (all apps) | `pnpm dev` |
| Dev (single app) | `pnpm --filter web dev` or `pnpm --filter docs dev` |
| Build all | `pnpm build` |
| Lint all | `pnpm lint` |
| Type-check all | `pnpm check-types` |
| Format | `pnpm format` |
| Generate UI component | `pnpm --filter @repo/ui generate:component` |

There are no tests yet in this repo.

## Architecture

Turborepo monorepo with two apps and shared packages.

```
apps/
  web/    — main React Router v7 app (port 3000)
  docs/   — Astro/Starlight docs site (port 3001)
packages/
  ui/               — shared React component library (@repo/ui)
  eslint-config/    — shared ESLint configs (@repo/eslint-config)
  typescript-config/ — shared tsconfig bases (@repo/typescript-config)
```

### web app (React Router v7)

Uses React Router v7 with Vite and file-system routing (Remix v2 conventions):

- `app/root.tsx` — root layout/document shell (exports `Layout`, default `App`, `ErrorBoundary`, `meta`, `links`)
- `app/routes.ts` — route config via `flatRoutes()` from `@react-router/fs-routes`
- `app/routes/` — file-system routes; `_index.tsx` = `/`, `login.tsx` = `/login`, nested via dots/underscores
- SSR enabled (`ssr: true` in `react-router.config.ts`)
- Route type generation: `react-router typegen` writes to `.react-router/types/`
- Import alias: `@/` maps to the workspace root of `apps/web/`

### Shared UI package

`packages/ui` exports components directly from source via the `exports` field (`"./*": "./src/*.tsx"`). Components are consumed as `@repo/ui/button`, `@repo/ui/card`, etc. — no build step required for the package itself.

### ESLint

ESLint is configured with `--max-warnings 0` in both apps and the UI package, so any warning is treated as an error.

### TypeScript

`check-types` runs `react-router typegen && tsc --noEmit` in the web app (generates route types before checking). The `@repo/typescript-config` package exposes presets: `base`, `nextjs`, `react-library`, and `vite`.
